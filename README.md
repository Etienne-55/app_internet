# Meu Portfólio

## Contexto
Site de portfólio pessoal para apresentar projetos e habilidades técnicas. Domínio: portfólio de desenvolvedor front-end.

## Público-alvo
- Perfil: recrutadores e tech leads, 25–40 anos
- Dispositivo principal: desktop, com suporte a mobile

## Problema
Desenvolvedores iniciantes não conseguem mostrar seus projetos de forma profissional fora do GitHub.

## Critério de sucesso
O usuário consegue ver o nome, projetos e contato em menos de 30 segundos sem rolar a página.

## Paleta — "Azul Confiança"
| Variável | Hex | Uso |
|---|---|---|
| --color-primary | #2563EB | Botões e destaques |
| --color-text | #1E293B | Texto principal |
| --color-bg | #F8FAFC | Fundo |
| --color-border | #E2E8F0 | Bordas |
| --color-success | #16A34A | Feedback positivo |

Contraste: #2563EB sobre branco = 4.6:1 — WCAG AA aprovado
